Chelsea Torres - Personal Portfolio Website
Overview

This project is a personal portfolio website developed as part of a course requirement. The website serves as a digital profile that showcases my educational background, technical skills, projects, internship experiences, and career aspirations in the field of Computer Science.

The design and content of this website were inspired by my existing portfolio and enhanced using Artificial Intelligence tools to improve structure, design, and user experience.

Objectives
Present a professional online profile.
Demonstrate web development skills.
Showcase academic and personal projects.
Apply Git version control practices.
Deploy a live website using GitHub Pages.
Explore the practical use of Artificial Intelligence in software development.

All generated outputs were reviewed, modified, tested, and implemented by the developer.
